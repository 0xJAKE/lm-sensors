  • I²C and SMBus implementations
    • Intel PIIX4 (82371AB)

      • Intel SMBus adapter, integrated with many other functions. There are datasheet for most chipsets.
    • VIA South Bridge (VT85C586B)

      • VIA I²C adapter, integrated with many other functions. There is a separate section with datasheets.
    • Acer Labs M1533/M1543C

      • Acer Labs SMBus adapter, integrated with many other functions.

  • Other protocol information
    • IPMI documentation

      • Intelligent Platform Management Interface, a standard for hardware monitor chip reading
  • Hardware Monitoring Applications using the lm_sensors package
    • Procmeter

      • A comprehensive system monitoring package. Nice desktop app. Recommended.
    • A web-based hardware monitor

      • This is created by Phil Edelbrock, one of our main project contributors
    • wmlm78/wmsensors homepage

      • wmlm78 and wmsensors are graphical extensions of lm_sensors, which can be docked in Window Manager.
    • klm

      • Brendon Humphrey's KDE display program for lm_sensors.
    • Sensors Server

      • Nice client/server application with a html-embedded Java applet.
    • GLS

      • Stijn Jonker's program to make nice graphs of the lm_sensors output. Recommended. Requires mysql package.
  • Other Hardware monitoring implementations (not using lm_sensors)

    • Ronald Schmidt's lm78 module

      • Written by Erik Hendriks. This Nasa site may have a newer version. There is also SCMS Cluster Management for Beowulf clusters available, which supports the LM78 through the above kernel module.
    • txwd, another lm78 module
    • András Major's LM75 module, and graphical monitor for it
    • Solaris drivers for the LM78
